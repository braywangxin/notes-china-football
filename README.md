# notes-china-football

本仓库用于归档和发布多个独立 HTML 页面，内容围绕中国足球领域相关的信息整理与记录。所有页面均为独立 HTML 文件，不依赖特定运行环境，可直接在浏览器中打开浏览。

## 项目简介

本项目作为中国足球相关信息的公开笔记与页面存档，收集、整理并发布一系列独立的 HTML 页面。这些页面可能包含赛事记录、数据整理、资料汇总或观察笔记等，旨在以静态网页形式提供阅读与参考。

仓库本身不针对任何特定网站或域名，所有页面内容均为独立编写与维护。

## 目录说明

```
notes-china-football/
├── docs/               # 用于 GitHub Pages 发布的 HTML 页面目录
│   └── *.html          # 各独立页面文件
├── assets/             # 样式、图片、脚本等静态资源（如有）
├── README.md           # 本文件
└── LICENSE             # 许可证文件
```

- `docs/`：存放所有公开发布的 HTML 页面，可通过 GitHub Pages 直接访问。
- `assets/`：存放页面引用的资源文件，按需组织。
- 每个 HTML 页面均为独立完整文件，部分页面可能引用 `assets/` 下的资源。

## 页面归档说明

- 每个 HTML 页面均独立归档，内容互不依赖。
- 页面命名尽量反映其主题或内容，例如 `match-notes-2024.html`、`data-overview.html` 等。
- 归档内容可能随时间更新、修正或新增页面，不保证所有页面持续同步更新。
- 部分页面可能包含时间戳或版本标记，以供参考。
